# Randomly card shuffle
Card Group :
1) Spade = S
2) Heart = H
3) Diamond = D
4) Club = C

Each group contain 13 cards which follow requirement:
1)  Number 2 to 9 as it is.
2)  1=A
3)  10=X
4)  11=J
5)  12=Q
6)  13=K


Card need to be shuffle randomly to n number of people.

PHP version 7.4.2
Laravel 8
